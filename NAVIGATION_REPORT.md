### Module-oriented flow
- `index.php` / `login_process.php` -> `dashboard.php` -> setting modules and managers.
- Settings pages (school/staff/fees/account/library/transport/exam) -> manager/list pages -> add/edit/delete/entry process pages -> back to manager/list via redirects.
- Search/result flows: `*_searchby_name.php` -> `*_search_result.php` or detail/receipt pages.
- Receipt/report flows: `entry_*` pages -> `*_reciept*.php` / `*_report*.php`.
- Student lifecycle: admission (`admission.php`, `add_admission.php`, `edit_admission.php`) -> fees/transport/library/exam/TC pages.
- Common includes from `includes/header.php`, `includes/sidebar.php`, `includes/footer.php`, plus module sidebars.
